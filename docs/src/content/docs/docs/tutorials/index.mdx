The tutorials are a problem-first learning rail. Each one opens with a scenario,
walks you into the wall it solves, then hands off to the next. Planned rail:

1. First run: from clone to a running harness and a first chat.
2. Run a workflow: the bundled smoke test, watched end to end.
3. Install a rib: add a package, see it light up a surface.
4. Build your first rib: scaffold a `@keelson/rib-*` package against the contract.

The capstone, building your first rib, is the one most readers come for.
