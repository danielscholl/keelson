The reference tier is the contract: precise, terse, and stable. Planned pages:

- The Rib contract: the `Rib` interface, its lifecycle hooks, the validation
  rules the harness enforces, and the discovery and activation sequence. The
  centerpiece, and the first reference page to land.
- Snapshots and surfaces: the streaming substrate ribs publish to.
- Workflow nodes: the seven-node taxonomy, the DAG features, and the substitution
  model.
- The CLI: commands, flags, and the stable exit-code contract.
- Providers: the `IAgentProvider` interface and the provider matrix.
