Keelson is a single-user agent harness. It wraps a coding agent with persistent
state, deterministic YAML workflows, and a typed extension contract. The harness
is the keel, the part that ships. The capabilities come from the ribs you bring.

These docs are tiered by audience. Start with Concepts to learn the shape of the
system, follow the Tutorials to build, and reach for the Reference when you need
the contract.
